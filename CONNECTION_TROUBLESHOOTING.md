# 接続トラブルシューティング

## ✅ サーバー状態確認

- **ポート3000**: LISTENING状態 ✅
- **Node.jsプロセス**: 実行中 ✅
- **サーバー**: 起動済み ✅

## 🔍 問題の原因

DNS_PROBE_FINISHED_NXDOMAINエラーは、ブラウザが`localhost`を解決できないことを示しています。

## 🔧 解決方法

### 方法1: 127.0.0.1を使用（推奨）

ブラウザで以下のURLにアクセスしてください：

```
http://127.0.0.1:3000
```

`localhost`ではなく`127.0.0.1`を使用することで、DNS解決を回避できます。

### 方法2: ブラウザのキャッシュクリア

1. **Chrome/Edgeの場合**:
   - `Ctrl + Shift + Delete` を押す
   - 「キャッシュされた画像とファイル」を選択
   - 「データを削除」をクリック

2. **別のブラウザで試す**:
   - Chrome、Edge、Firefoxなど別のブラウザで試す

### 方法3: hostsファイルの確認

`C:\Windows\System32\drivers\etc\hosts` ファイルを確認：

```
127.0.0.1       localhost
::1             localhost
```

この行が存在しない、またはコメントアウトされている場合は追加してください。

### 方法4: ファイアウォールの確認

WindowsファイアウォールがNode.jsをブロックしていないか確認：

1. Windowsセキュリティ → ファイアウォールとネットワーク保護
2. 「アプリがファイアウォールを通過することを許可」を確認
3. Node.jsが許可されているか確認

## 📋 確認手順

### 1. サーバーログの確認

サーバーを起動したPowerShellウィンドウで、以下のようなメッセージが表示されているか確認：

```
ready - started server on 0.0.0.0:3000, url: http://localhost:3000
```

### 2. エラーログの確認

サーバーのログにエラーメッセージがないか確認してください。

### 3. 別の方法でアクセス

- `http://127.0.0.1:3000` を試す
- `http://localhost:3000` を試す
- 別のブラウザで試す

## 🎯 推奨アクション

**今すぐ試してください**：

1. ブラウザで `http://127.0.0.1:3000` にアクセス
2. これでアクセスできない場合は、サーバーのログを確認
3. エラーメッセージがあれば、その内容を共有してください
